Lists

There Are 3 Types of Lists :

1. Ordered List
2. Unordered List
3. Description List

1. Ordered List


My TO DO list is:

  1. Chemistry Assignment
  2. C language Works:
    1. Write 5 Codes on Pointers.
    2. Assignment
  3. Maths Do Home Work 10 Problems

2. Unordered List


My TO DO list is:

  • Chemistry Assignment
  • C language Works:
    • Write 5 Codes on Pointers.
    • Assignment
  • Maths Do Home Work 10 Problems

3. Description List


My TO DO list is:

Chemistry Assignment
C language Works:
- Write 5 Codes on Pointers.
- Assignment
Maths Do Home Work 10 Problems
